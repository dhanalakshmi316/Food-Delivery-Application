**Food Delivery Application**

## Overview

The **Food Delivery Application** is a Java Full Stack web application designed to provide a seamless online food ordering experience for customers while enabling restaurants to efficiently manage their menus and orders. The application offers a user-friendly interface where customers can browse restaurants, explore menus, add food items to a shopping cart, place orders, and track their order status.

The system follows a three-tier architecture consisting of a responsive frontend, a robust backend, and a relational database. It ensures secure user authentication, efficient order processing, and reliable data management, making it suitable for real-world food delivery services.

## Key Features

* User registration and secure login authentication
* Browse restaurants and food categories
* View detailed menus with pricing and descriptions
* Search and filter food items
* Add, update, and remove items from the shopping cart
* Place orders with delivery details
* Order confirmation and order history
* Restaurant dashboard for menu and order management
* Admin panel for managing users, restaurants, and food items
* Responsive design for desktop and mobile devices
